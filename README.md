> # Servlet-Project-PMS Project
서블릿 + JSP 프로젝트 협업 프로그램  
<br/>

> ## 🖥 프로젝트 소개
회사 내부의 프로젝트 진행 시 여러 기능을 통해 프로젝트를 더욱 효율적으로 진행, 관리 할 수 있는 협업 프로그램.  
<br/>


> ## 📅 개발기간
* 23.05.26일 ~ 23.06.09일
<br/>
  
### ⚙ 개발 환경
* `Java 8`
* `JDK 1.8.0`
- **IDE** : Eclipse
- **Database** : Oracle DB(11xe)
</br>

### 📌 주요 기능
> 메인 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/0c71a1d0-3106-4b06-8be9-3708af7868af"/>
  
📋 구현 기능

<br/>

> 프로젝트 등록 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/7fb649d1-f711-4d61-91d2-017f824e77d3"/>
  
📋 구현 기능  
* 프로젝트를 등록 할 수 있는 페이지

<br/>

> wbs 차트 출력 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/4177e073-4004-4788-a776-b87951b881f8"/>
  
📋 구현 기능  
* 해당 프로젝트의 진척도를 한 눈에 볼 수 있게끔 차트로 출력되는 페이지

<br/>

> 본인의 작업 목록과 완료율 출력 및 수정 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/6f08c78b-98b8-47f5-9804-38fe5e66936e"/>
  
📋 구현 기능  
* 작업 관리 탭에서 내작업 세부 탭을 클릭하면 본인이 진행한 작업 목록이 출력되고 완료율을 입력, 수정 할 수 있는 페이지
* 완료율이 100%가 되면 '진행중' 문구가 '완료'로 바뀌도록 구현
