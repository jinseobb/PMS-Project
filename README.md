> # Servlet-Project-PMS Project
서블릿 + JSP를 활용하여 제작한 협업 프로그램  
<br/>

> ## 🖥 프로젝트 소개
회사 내부의 프로젝트 진행 시 여러 기능을 통해 프로젝트를 더욱 효율적으로 진행, 관리 할 수 있는 협업 프로그램.  
<br/>


> ## 📅 개발기간
* 23.05.26일 ~ 23.06.09일
<br/>
  
> ## ⚙ 개발 환경
* `Java 8`
* `JDK 1.8.0`
- **IDE** : Eclipse
- **Database** : Oracle DB(11xe)

### 📌 주요 기능
> 메인 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/0c71a1d0-3106-4b06-8be9-3708af7868af"/>
  
📋 구현 기능  
* 메인 화면으로 본인의 프로젝트 세부 현황에 대한 정보가 출력되는 페이지

<br/>

> 프로젝트 등록 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/7fb649d1-f711-4d61-91d2-017f824e77d3"/>
  
📋 구현 기능  
* 프로젝트를 등록 할 수 있는 페이지

<br/>

> wbs 차트 출력 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/4177e073-4004-4788-a776-b87951b881f8"/>
  
📋 구현 기능  
* 해당 프로젝트의 진척도를 한 눈에 볼 수 있게끔 차트로 출력되는 페이지

<br/>

> 본인의 작업 목록과 완료율 출력 및 수정 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/6f08c78b-98b8-47f5-9804-38fe5e66936e"/>
  
📋 구현 기능  
* 작업 관리 탭에서 내작업 세부 탭을 클릭하면 본인이 진행한 작업 목록이 출력되고 완료율을 입력, 수정 할 수 있는 페이지
* 완료율이 100%가 되면 '진행중' 문구가 '완료'로 바뀌도록 구현

<br/>

> 팀원의 작업 진척도 출력 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/38dc98e7-f765-4a7b-b6e7-2e864ad0a3ba"/>
  
📋 구현 기능  
* 작업 관리 탭에서 일반 작업 세부 탭을 클릭하면 팀원이 진행하는 작업 목록과 진척도를 도넛 차트로 출력되는 페이지

<br/>

> 프로젝트 산출물 목록 출력 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/1af67c86-5e2e-42a9-b8c1-d1ef378ce563"/>
  
📋 구현 기능  
* 해당 프로젝트에서 발생한 모든 산출물 목록이 출력되고, 상세정보 확인 및 다운로드가 가능한 페이지

<br/>

> 개인 및 팀원 일정 출력 화면
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/f6d3be07-5647-420e-8f74-4146eb01a809"/>
  
📋 구현 기능  
* 모든 팀원의 일정을 확인 할 수 있는 페이지
* 팀원 전체의 일정도 확인 가능 하며, 확인하고 싶은 팀원을 선택하여 선택한 팀원 개인의 일정 역시 확인 가능하다.

<br/>

> 결재 안건 관리 페이지
<img width="350px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/e454c780-b2d4-48d6-9965-100f6f254499"/>
<img width="350px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/98944668-8b52-416e-a659-e98b67dd73c2"/>
  
📋 구현 기능  
* PA는 PM에게 결재를 올릴 수 있다.
* PM은 PA가 올린 결재 안건을 관리 할 수 있다.

<br/>

> 이슈 사항 관리 페이지
<img width="550px" src="https://github.com/jinseobb/PMS-Project/assets/131458472/f1280002-c603-4a7e-b384-650e828fbf89"/>
  
📋 구현 기능  
* PA는 PM에게 이슈 사항 처리를 요청 할 수 있다.
* PM은 PA가 요청한 이슈 사항에 대한 처리를 할 수 있다.
